## Backend

- Framework: [echo](https://echo.labstack.com/)
- Database Driver: [pgx](https://github.com/jackc/pgx)
- Logging: [zerolog](https://github.com/rs/zerolog)
- Validator: [validator](https://github.com/go-playground/validator)
- Config Management: [koanf](https://github.com/knadh/koanf)
- Migrator: [tern](https://github.com/jackc/tern)
- Tests: [testify](https://github.com/stretchr/testify)
- Database Container For Testing: [testcontainers](https://testcontainers.com/)
